Nav demo

Super simple navigation. Loosely inspired by Google’s Material Design. Designed and implemented by Cassidy James, based off of his ideas for the System76 admin site.

Basic Idea

Nav uses an HTML <nav> element as the container, with child <ul>s of <li>s with <a>s. The last <ul> gets right-aligned, i.e. for login or search. For example, this page’s nav looks like:

<nav>
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#" class="active">Section 1</a></li>
    <li><a href="#">Section 2</a></li>
    <li><a href="#">Section 3</a></li>
    <li><a href="#">Section 4</a></li>
  </ul>
  <ul>
    <li><a href="#">Log In</a></li>
    <li><a href="#">GitHub</a></li>
  </ul>
</nav>

In this demo, purple is a placeholder for the main color and red is an accent color.

.active links

Any a.active links will get an accent-colored underline applied.

@media queries

There is one media query in the CSS: @media (min-width: 768px) { … } which simply sets a bit higher padding on the links for larger displays and sets up…

position: sticky

I’m pulling in a (completely optional) polyfill for position: sticky. In supported browsers, that makes the nav bar stay in the normal flow until you scroll, at which point it effectively becomes position: fixed. Only enabled on larger displays.
